# Cloudflare Pages POC
A proof of concept for using Cloudflare Pages with Hono & React

## Built using
Cloudflare Pages, Hono, Vite, React, TailwindCSS, TypeScript, Turso, Drizzle, Zod

## Get Started
- ```npm install```
- Create a .env file for environment variables
- Create a .dev.vars file for Cloudflare environment variables
- Run ```npm run watch``` in one terminal
- Run ```npm run dev``` in another terminal
